# Feynman Technique

使用费曼学习法帮助用户深入理解和掌握任意概念。

## 触发方式

当用户表达以下意图时激活：
- 学习某个概念
- 用费曼法解释
- 简单讲清楚 / 教给我
- 用最简单的话说
- "费曼一下" / "以教代学" / "简化概念"

## 适用场景

- 解释技术概念（如 HTTP 协议、区块链、机器学习等）
- 梳理知识盲区
- 对复杂主题建立直觉理解

## 方法核心

1. **选择一个概念** — 确定要学习的目标概念
2. **用简单的语言解释** — 用自己的话讲清楚，如同教给一个完全不懂的人
3. **识别知识缺口** — 发现解释不清楚的地方，返回原始材料重新学习
4. **简化和类比** — 用类比和精简的语言深化理解
